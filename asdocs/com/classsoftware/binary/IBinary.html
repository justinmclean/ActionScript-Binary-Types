Packagecom.classsoftware.binary
Interfacepublic interface IBinary
Implementors Binary

Binary type interface. Use when a variable can have different binary types (eg a Byte or Word). 
     var timer:IBinary;
     
     if (largeTimer)
         timer = new Word();
     else
         timer = new Byte();
     
     // later in the code
     timer.inc();
